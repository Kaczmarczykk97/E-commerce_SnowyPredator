#E-commerce: Snowy Predator

Opis:

Snowy Predator - Store to projekt strony internetowej stworzony przy użyciu HTML5, CSS3, JavaScript (ES6) oraz React z użyciem React Router. Celem projektu jest imitacja sklepu internetowego z bogatym zestawem funkcji, które są charakterystyczne dla prawdziwych sklepów online.

## Funkcje projektu:

1. **Strona Główna:**
   - Przejrzysty układ prezentujący najnowsze produkty i promocje.
   - Szybki dostęp do głównych kategorii produktów.
   - Informacje o personelu
   - Informacje o eventach
   - Video promujące

2. **Strony Produktów:**
   - Szczegółowe strony dla każdego produktu z opisem, parametrami i opiniami.
   - Dodawanie produktów do koszyka zakupowego.

3. **Koszyk Zakupowy:**
   - Zarządzanie zawartością koszyka, modyfikowanie ilości produktów.
   - Całkowity koszt zakupów i proces składania zamówienia.

4. **Strony Dodatkowe:**
   - O nas, Kontakt, Regulamin, Polityka Prywatności.
   - Łatwy dostęp do informacji o sklepie.
   - Imitacja strony logowania

5. **Responsywność:**
   - Strona jest dostosowana do różnych urządzeń, zapewniając użytkownikom komfortową nawigację na smartfonach, tabletach i komputerach.

## Technologie:

- HTML5
- CSS3
- JavaScript (ES6)
- React
- React Router

## Instalacja i Uruchomienie:

Aby uruchomić lokalnie, wykonaj następujące kroki:

1. Sklonuj repozytorium: `git clone https://github.com/Kaczmarczykk97/E-commerce_SnowyPredator.git`
2. Przejdź do folderu projektu: `cd E-commerce_SnowyPredator`
3. Zainstaluj zależności: `npm install`
4. Uruchom projekt: `npm start`
